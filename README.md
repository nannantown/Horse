# 🏇 G1 Turf Analytics

JRA G1レースの過去記録・名馬データ・統計分析を、スマホで見やすく確認できるWebアプリ。

## 機能

- **ホーム**: 次のG1までのカウントダウン、開催状況サマリ
- **G1レース一覧**: 年間24レースの開催日・コース・距離。各レース詳細では脚質傾向・1番人気勝率・過去優勝馬を表示
- **名馬データベース**: イクイノックス／ドウデュース／アーモンドアイ／ディープインパクト等の代表馬
  - 能力レーダーチャート (スピード/スタミナ/パワー/ダート/メンタル)
  - 血統 / 戦績テーブル
- **統計分析**: G1リーディング騎手・種牡馬・競馬場別開催数・距離分布のグラフ可視化
- **予想スコア**: 人気・脚質・前走・コース適性から100点満点でスコアリング (S〜D評価)

## モバイル最適化

- 下部タブナビゲーション
- セーフエリア対応 (iPhone)
- ダーク/ライトテーマ切替 (設定保存)
- PWA対応 (ホーム画面に追加可能)
- タッチ最適化UI

## 使い方

`index.html` をブラウザで開くだけ。バックエンド不要のフロントエンド完結アプリ。

```sh
# ローカルサーバで開く (推奨)
python3 -m http.server 8000
# → http://localhost:8000 にアクセス
```

スマホで見るときは同一Wi-Fi上で `http://<PCのIP>:8000` にアクセス。

## データについて

掲載のレース日程・過去結果・馬データは公開されているJRA公式記録に基づく。予想スコアは過去傾向の簡易シミュレーションであり、実際の馬券購入の参考にはしないこと。

## ファイル構成

```
index.html     # エントリポイント
styles.css     # モバイルファーストCSS
app.js         # SPAロジック (5ビュー: ホーム/G1/名馬/分析/予想)
data.js        # レース・馬・騎手・種牡馬データセット
manifest.json  # PWAマニフェスト
icon.svg       # アプリアイコン
```
